# Internship Finder

A responsive internship listing web application built using React.js and Tailwind CSS.  
The application allows users to browse internships and filter them based on different options like profile, location, duration, and stipend.

The project is inspired by internship/job platforms and focuses on clean UI, responsive design, and reusable React components.

---

## Features

- View internship listings
- Filter internships by:
  - Profile
  - Location
  - Duration
  - Stipend
- Pagination support
- Responsive design for mobile and desktop
- Loading and error handling
- Smooth scroll to top on page change

---

## Tech Used

- React.js
- JavaScript
- Tailwind CSS

## How It Works

- Internship data is fetched from an API
- Filters update the internship list dynamically
- Pagination is used to show limited internships per page
- On changing pages, the screen scrolls back to the top automatically
